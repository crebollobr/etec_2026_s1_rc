
# 🌐 Infraestrutura de Redes: Topologias e Meios de Transmissão

Documentação técnica sobre a organização física de redes e as estratégias de propagação de dados em camadas lógicas.

---

## 🏗️ Topologias de Rede

A disposição física e lógica dos nós em uma rede determina a resiliência e a escalabilidade do sistema.

### 1. Topologias Físicas Comuns

| Topologia | Funcionamento | Vantagem | Desvantagem |
| --- | --- | --- | --- |
| **Estrela (Star)** | Todos os nós conectam-se a um hub/switch central. | Fácil diagnóstico e isolamento de falhas. | Ponto único de falha (o nó central). |
| **Barramento (Bus)** | Todos os nós compartilham um único meio de transmissão. | Baixo custo de implementação. | Colisões altas; se o cabo rompe, a rede para. |
| **Anel (Ring)** | Cada nó conecta-se a outros dois, formando um círculo. | Transmissão determinística. | Um nó falho pode derrubar o anel inteiro. |
| **Malha (Mesh)** | Conexões redundantes entre múltiplos nós. | Altíssima tolerância a falhas. | Custo e complexidade de cabeamento elevados. |
| **Híbrida** | Combinação de duas ou mais topologias. | Flexibilidade total. | Gerenciamento complexo. |

## ⚡ Meios de Transmissão Lógicos

Diferente do meio físico (cabos), o meio lógico define como os dados são colocados e interpretados na rede.

### 1. Métodos de Acesso ao Meio (MAC)

Regras que ditam quando um dispositivo pode transmitir dados para evitar colisões.

* **CSMA/CD (Carrier Sense Multiple Access with Collision Detection):**
* Utilizado em redes Ethernet.
* O dispositivo "ouve" o meio; se houver colisão, ele para e tenta novamente após um tempo aleatório.


* **CSMA/CA (Carrier Sense Multiple Access with Collision Avoidance):**
* Padrão para redes Wi-Fi (IEEE 802.11).
* Tenta evitar a colisão antes que ela ocorra através de sinais de confirmação (RTS/CTS).


* **Token Passing:**
* Um "token" digital circula na rede; apenas quem possui o token pode transmitir.



### 2. Modos de Transmissão

Define a direção do fluxo de dados logicamente.

> [!NOTE]
> A escolha do modo depende da necessidade de interatividade e da largura de banda disponível.

1. **Simplex:** Transmissão em apenas uma direção (ex: TV/Rádio).
2. **Half-Duplex:** Transmissão em ambas as direções, mas **não** simultaneamente (ex: Walkie-talkie).
3. **Full-Duplex:** Transmissão e recepção simultâneas (ex: Redes Ethernet modernas e Telefonia).

## 🛠️ Comandos de Diagnóstico de Rede

Exemplos de comandos para validar a topologia lógica e conectividade:

```bash
# Verifica a configuração de interface e endereçamento lógico
ip addr show   # Linux
ipconfig       # Windows

# Rastreia a rota lógica até o destino (Topologia de saltos)
traceroute google.com  # Linux/macOS
tracert google.com    # Windows

# Exibe a tabela ARP (mapeamento Lógico IP -> Físico MAC)
arp -a

```
