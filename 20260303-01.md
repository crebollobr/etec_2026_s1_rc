## 1. Fundamentos: Camadas de Rede (TCP/IP)

Diferente do modelo teórico OSI (7 camadas), o mercado utiliza o **Modelo TCP/IP** para explicar o fluxo de dados.

| Camada | Função Principal | Protocolos Comuns |
| --- | --- | --- |
| **Aplicação** | Interface com o usuário/software. | HTTP, DNS, FTP, SMTP |
| **Transporte** | Comunicação ponta a ponta e controle de fluxo. | TCP, UDP |
| **Internet** | Endereçamento lógico e roteamento de pacotes. | IP (IPv4/IPv6), ICMP |
| **Acesso à Rede** | Conversão de bits em sinais (elétrico/luz/rádio). | Ethernet, Wi-Fi, ARP |

## 2. Hardware: Switch vs. Roteador

Entender a diferença é entender em qual camada o dado está sendo processado.

### 🔄 Switch (O "Inteligente" da LAN)

* **Camada:** 2 (Enlace).
* **Endereçamento:** Usa a tabela **MAC Address**.
* **Função:** Conecta dispositivos dentro da mesma rede local.
* **Vantagem:** Cria canais dedicados entre portas, evitando colisões.

### 🌐 Roteador (O "Guarda Rodoviário" da WAN)

* **Camada:** 3 (Rede).
* **Endereçamento:** Usa o endereço **IP**.
* **Função:** Conecta redes diferentes (Ex: Sua casa com a Internet).
* **Vantagem:** Determina a melhor rota para o pacote chegar ao destino.

## 3. Topologias de Rede

A forma como os nós são interconectados define a resiliência e o custo da rede.

### 🏗️ Topologias Físicas

* **Estrela (Star):** Todos os cabos convergem para um ponto central (Switch).
> *Padrão atual: Se um PC falha, a rede continua.*


* **Barramento (Bus):** Todos compartilham um único cabo (backbone).
> *Obsoleto: Se o cabo principal rompe, tudo para.*


* **Anel (Ring):** Cada computador é conectado ao próximo, formando um círculo.
* **Malha (Mesh):** Conexões redundantes entre vários pontos.
> *Usado em infraestruturas críticas e Wi-Fi Mesh.*



### ⚡ Meios de Transmissão Lógicos

Define como os dados fluem pelo meio físico:

1. **Simplex:** Apenas um sentido (Ex: Teclado -> PC).
2. **Half-Duplex:** Ambos sentidos, mas um por vez (Ex: Hubs antigos ou Rádio amador).
3. **Full-Duplex:** Ambos sentidos simultaneamente (Ex: Switch moderno).
